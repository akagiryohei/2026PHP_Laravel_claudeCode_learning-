# Laravel学習プロジェクト概要

## プロジェクト情報

- **作成日**: 2026-05-11
- **目的**: PHPとLaravelの知識再習得・強化
- **題材**: タスク管理アプリ

## 学習者プロフィール

- エンジニア歴: 約3年
- PHP/Laravel経験: 約1年（飛び飛びの実務経験あり）
- 最終使用: 約1年前
- 現状: 基礎（ルーティング、変数宣言レベル）から再学習が必要

## 技術スタック

- **バックエンド**: PHP 8.1 / Laravel 10
- **フロントエンド**: Blade（後でデザインを探して適用予定）
- **DB**: MySQL 8.0
- **認証**: Laravel Breeze（Blade）

## 学習フェーズ

| Phase | テーマ | 主な学習内容 | ステータス |
|-------|--------|-------------|-----------|
| 1 | Laravel基礎 | ルーティング、MVC、認証、Blade | **完了** |
| 2 | CRUD完成 | Eloquent、バリデーション | **完了** |
| 3 | リレーション | 1対多、クエリビルダー | 未着手 |
| 4 | 応用 | 検索、認可、最適化 | 未着手 |

## 進め方の方針

1. 各Phaseは実装しながら解説を受ける形で進める
2. フロントエンドはPhase完了後にデザインを取得して後付けで整える
3. 詰まった部分は都度解説を求める
