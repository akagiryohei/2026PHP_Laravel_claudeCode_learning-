# Phase 3: リレーション — カテゴリ・期限・優先度

## 目標

テーブル間のリレーションを理解し、カテゴリ・期限・優先度機能を追加する。

## 実装する機能

- [ ] カテゴリの追加・管理
- [ ] タスクへのカテゴリ紐付け
- [ ] 期限日の設定
- [ ] 優先度（高/中/低）の設定

## 学習ポイント

### Eloquentリレーション
- 1対多（`hasMany` / `belongsTo`）
- リレーション経由のデータ取得（`$task->category->name`）
- Eager Loading（`with()`）でN+1問題を防ぐ

### マイグレーション（応用）
- 外部キー制約（`foreignId()->constrained()`）
- 既存テーブルへのカラム追加

### クエリビルダー
- `where()`, `orderBy()`, `latest()`
- スコープ（`scopeCompleted()` など）
